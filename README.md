# BBC News Text Classifier with LSTM using TensorFlow 2.0
 
 Classify BBC News by using LSTM using Tensorflow 2.0

BBC News are labeled with the following categories 
- Entertainment
- Sport
- Politics
- Business
- Tech

To train write **python learn.py**


Current Performance 

1. Train Accuracy: 95.0%
2. Validation Accuracy: 93.5%
3. Test Accuracy: 91%
